# kk 音标

音标：标明发音

kk 音标分为：母音和子音；母音可以单独存在，子音必须配母音

## 母音

重点：	[ə]	[ɚ] 轻声（懒人音），其余母音**四声**（重音）

- [ə] ：额
- [ɚ] ：儿，sister[ 美 /'sɪstɚ/] 姐妹，重音记号不在 ɚ 上

v 图与嘴形有关，开口就是张开很大；从扁嘴，到嘴巴张大

```
扁
[i]   \             /
[ɪ]    \           /
[e]     \         /
         \       /
          \     /
            ---
     大

```
